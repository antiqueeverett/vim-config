#### Usage

-   clone
```
git clone https://github.com/edisonslightbulbs/vim-dotfiles.git
```

-   source
```
# after clone
cd vim-dotfiles

# add source directive to .zshrc dotfile
echo "source $PWD/config.vim" >> ~/.vimrc
```

-   source `.zshrc`
```
source ~/.zshrc
```
