#### Usage

-   clone
```
# clone
git clone https://github.com/edisonslightbulbs/vim-dotfiles.git

# cd into the repo
cd vim-dotfiles

# append source directive in .vimrc
echo "source $PWD/source-files.vim" >> ~/.vimrc
```

and thats pretty much it!
